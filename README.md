Web-Audit-Automation-Framework (12-Point Audit)
This is a professional-grade web audit framework designed for high-performance testing and automated reporting.

🚀 Key Features:
12-Point Automated Audit: Covers everything from UI elements to broken link checks.

Sunday Auto-Report: A 'Set-and-Forget' feature that generates and emails full reports every Sunday.

Allure Reporting: Beautiful, visual graphs and dashboards for stakeholders.

Tech Stack: Selenium, Pytest, Python, and Allure.

🗓 Client-Centric Automation:
The framework includes a scheduler that triggers a full audit every Sunday. Results are automatically converted into PDF/HTML and emailed directly to the client's inbox.
